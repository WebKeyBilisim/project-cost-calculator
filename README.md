## 🧠 What is this?

**Project Cost Calculator** is a small and humorous web-based calculator that estimates the complexity level of a digital project.

It does **not** replace real project analysis.  
It does **not** magically know the client’s budget.  
It does **not** prevent scope creep.

But it does give a fun first impression about how “simple” a project really is.

> Client: “It is just a small website.”  
> Calculator: “Sure. With login, payment, admin panel, reports, API, mobile design and urgent delivery.” ☕

---

## 🎯 Why did we build this?

Because in software projects, the word **simple** is usually the beginning of an adventure.

This mini tool helps demonstrate that project complexity depends on multiple factors:

- Number of pages
- Admin panel requirement
- User login
- Payment integration
- API integration
- Reporting needs
- Live streaming features
- Multi-language support
- Deadline pressure
- And of course, coffee consumption

## ☕ Coffee-Driven Logic

The calculator uses a very scientific formula:

```js
complexity = features + integrations + deadlinePressure + coffeeRisk;
```

Then it translates the result into a business-friendly message:

| Score | Meaning |
|---|---|
| 🟢 Low | “This may actually be simple.” |
| 🟡 Medium | “We should write this down.” |
| 🟠 High | “This needs planning, testing and coffee.” |
| 🔴 Critical | “Call the team. And the coffee machine.” |
